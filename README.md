# CareerAI - Student Career Toolkit

A free AI-powered career toolkit built for students to prepare for jobs and internships.

## Features

- Resume Optimizer - Get ATS score and missing keywords
- Mock Interview - Practice with AI and get scored feedback
- Cover Letter Generator - Professional letters in seconds
- LinkedIn Bio Generator - Stand out to recruiters
- Skills Gap Analyzer - Know exactly what to learn next

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Groq API
- LLaMA 3 language model

## How to Run

1. Clone this repository
2. Open app.js and add your Groq API key
3. Open index.html in your browser
4. Start using all 5 tools

## Getting a Groq API Key

1. Go to console.groq.com
2. Sign up for free
3. Click API Keys and create a new key
4. Paste it in app.js where it says YOUR_GROQ_KEY_HERE
